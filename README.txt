Pour exécuter ce programme, lancez un terminal et tapez :
    make
Puis :
    ./main

Il n'y a qu'un seul scénario pour l'instant, celui de base
les variables liées aux accessoires sont à modifier dans le main.cpp
les ratios de personnalité sont à modifier dans milieu.h
les différentes variables liés aux capteurs sont à modifier dans concreteBestiole.h/.cpp
